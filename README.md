# Task Tracker API

A simple backend API built with **FastAPI** that allows you to create, read, update, and delete tasks. This project demonstrates full CRUD functionality, input validation with Pydantic, and a clean API structure.
This was created to get an understanding of CRUD and pydantic, as well as APIs as a whole. 

## **Features**

- Create tasks with a title and completion status
- Read all tasks
- Update tasks by ID
- Delete tasks by ID
- Input validation with Pydantic
- Interactive API documentation with Swagger UI

### **Requirements**

- FastAPI
- Uvicorn
- Pydantic

Install dependencies:

pip install fastapi uvicorn pydantic
